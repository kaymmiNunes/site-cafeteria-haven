# Cafeteria Haven

Site estático e responsivo desenvolvido para a **Cafeteria Haven**, uma cafeteria fictícia com identidade visual aconchegante, cardápio visual e navegação simples. O projeto apresenta uma proposta de página institucional para divulgação de produtos, ambiente, informações de contato e horário de funcionamento.

## Objetivo

Criar uma página web moderna, responsiva e funcional para uma cafeteria fictícia, demonstrando a estrutura de um site institucional com foco em apresentação visual, organização do cardápio e boa experiência de navegação para o usuário.

## Funcionalidades

* Página inicial com banner de destaque.
* Menu de navegação com acesso às principais seções.
* Cardápio visual com imagens, nomes e descrições dos produtos.
* Seção institucional sobre a cafeteria.
* Área de contato com endereço, telefone e horário de funcionamento.
* Layout responsivo para celulares, tablets e desktops.
* Menu interativo controlado por JavaScript.
* Estrutura simples e organizada para estudos de front-end.
* Paleta visual acolhedora e adequada ao tema cafeteria.
* Imagens utilizadas para reforçar a identidade visual do projeto.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript Vanilla
* Gemini para criação de imagens
