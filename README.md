# ☕ Cafeteria Haven

Um site simples e elegante desenvolvido para apresentar uma cafeteria fictícia, destacando cardápio, ambiente e identidade visual.

## 📌 Sobre o Projeto

Cafeteria Haven é um site estático construído com **HTML, CSS e JavaScript**, focado em design limpo, navegação simples e experiência visual agradável.  
O objetivo é demonstrar um layout moderno para uma cafeteria, podendo ser usado como base para portfólios, pequenos negócios ou estudos de front-end.

## 🚀 Funcionalidades

- Página inicial com destaque visual do banner
- Cardápio de cafés com imagens e descrições
- Navegação responsiva
- Menu interativo controlado por JavaScript
- Layout estilizado com paleta de cores acolhedora
- Suporte a dispositivos móveis

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estruturação da página
- **CSS3** – Estilização e responsividade
- **JavaScript (Vanilla)** – Menu interativo
- **Gemini** - Para a criação de imagens

## 📱 Responsividade

O layout foi planejado para se adaptar a diferentes tamanhos de tela, oferecendo boa navegação em:

- Celulares
- Tablets
- Desktops
